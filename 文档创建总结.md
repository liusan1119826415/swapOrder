## 🎯 重点内容：EasySwapBackend 部署

### 完整部署流程

#### 1. 准备工作
```bash
# 确保三个仓库在同一目录下
nft-market/
├── EasySwapBackend/
├── EasySwapBase/
└── EasySwapSync/
```

#### 2. 数据库设置（推荐 Docker）
```bash
# MySQL
docker run -d --name mysql-nft \
  -p 3306:3306 \
  -e MYSQL_ROOT_PASSWORD=your_password \
  -e MYSQL_DATABASE=easyswap_nft_market \
  mysql:8.0

# Redis
docker run -d --name redis-nft \
  -p 6379:6379 \
  redis:7-alpine
```

#### 3. 配置文件 (config/config.toml)
```toml
[server]
port = 8080
mode = "debug"  # 生产环境用 "release"

[database]
host = "localhost"
port = 3306
user = "root"
password = "your_password"
database = "easyswap_nft_market"

[redis]
host = "localhost"
port = 6379
password = ""
db = 0

[chains]
[[chainSupported]]
name = "sepolia"
chain_id = 11155111
rpc_url = "https://sepolia.infura.io/v3/YOUR_KEY"

[easyswap_market]
contract = "0xYourContractAddress"
```

#### 4. 启动服务
```bash
# 开发模式
cd EasySwapBackend/src
go run main.go

# 生产模式（编译二进制）
go build -o easyswap-backend src/main.go
./easyswap-backend

# 或使用 Systemd（Linux 生产环境）
sudo systemctl start easyswap-backend
```

#### 5. 验证 API
```bash
curl http://localhost:8080/api/v1/analytics
curl http://localhost:8080/api/v1/collections
curl http://localhost:8080/api/v1/activities
```

## 🎯 重点内容：EasySwapSync 部署

### 完整部署流程

#### 1. 配置同步服务
```bash
cd EasySwapSync
cp config/config.toml.example config/config.toml
```

#### 2. 编辑配置文件
```toml
[database]
host = "localhost"
port = 3306
user = "root"
password = "your_password"
database = "easyswap_nft_market"

[chain]
rpc_url = "https://sepolia.infura.io/v3/YOUR_KEY"
chain_id = 11155111

[contract]
address = "0xYourContractAddress"

[sync]
start_block = 0
batch_size = 1000
poll_interval = 15
```

#### 3. 启动同步
```bash
go run main.go daemon
```

#### 4. 验证同步状态
```sql
-- 检查合集数量
SELECT COUNT(*) FROM collections;

-- 检查活动记录
SELECT * FROM activities ORDER BY created_at DESC LIMIT 10;
```
